这便是 float 属性所带来的高度塌陷的现象，既然要实现文字环绕，那图片所在块中所占据的的高度肯定是要丢失的，不然文字都没法和图片齐平，还怎么实现环绕效果呢？

# 三、clear 属性

上面我们有提到 float 是具有导致包含块高度塌陷这一特性的，这一特性在有些实际应用中是开发者并不想看到的，往往开发者都期望包含块中的某个子元素在设置了 float 属性后，包含块还是能拥有设定的高度值。

所以基于这个情况，我们可以用到clear属性来清除浮动元素对其它元素带来的影响。

# 3.1 使用伪元素:after 清除（推荐）

.container:after {
  content: ' '; // 内容不能为空
  display: block;
  clear: both;
}
